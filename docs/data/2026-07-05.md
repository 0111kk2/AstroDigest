## 🚨 新天体・トランジェント速報(GCN 4報 / 3イベント)

### GRB 260704A (1報)
SVOM/ECLAIRsによって検出されたガンマ線バースト(GRB)に対し、Einstein Probe (EP) のX線望遠鏡 (FXT) が迅速な追観測を実施しました。GRB発生の約1時間後に開始された約2.6 ksの露光で、R.A., Dec. = 318.7151, -25.5074 deg (J2000) に位置し、誤差約10 arcsecの未登録X線源が発見されました。このX線源の平均非吸収フラックスは1.49 x 10^(-12) erg/s/cm^2 (0.5-10 keV) と測定されており、ECLAIRsおよびMXTの誤差円内に収まっています。

### GRB 260629A (2報)
このイベントは短いガンマ線バースト(GRB)であり、Fermi/GBM、AstroSat CZTI、SVOM/GRM、GECAM-Aという複数の高エネルギー衛星によって同時に検出されました。SVOM/GRMは15-5000 keV帯でT90が約1.2秒の単一パルスを検出し、GECAM-Aは70-6000 keV帯でT90が約3.2秒の単一パルスを観測しました。これらの観測から、短いガンマ線バーストとして特徴づけられており、各衛星はイベントの光度曲線データを取得しています。

### IceCube-260704A (1報)
IceCubeは2026年7月4日07:16:58.19 UTに、天体起源である中程度の確率を持つ高エネルギーニュートリノ候補（トラック型イベント）を検出しました。このアラートの偽警報率は年間0.8177イベントと推定されており、IceCube検出器は正常に動作していました。初期の自動警報後、より洗練されたオフライン解析が適用され、ニュートリノの方向が改善されましたが、現時点での他の追観測の報告はありません。
