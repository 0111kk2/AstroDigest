-   **目的**: 第4世代シンクロトロンであるアップグレードされたAdvanced Photon Source (APS-U)における超高速X線材料ダイナミクス実験用の、より高速かつ高輝度なシンチレータの開発。
-   **方法・データ**: 市販のLaBr3とCeBr3を含む様々なシンチレータ材料を評価。減衰時間、光収率、吸湿性、光学結合効率、量子効率などの特性を評価。材料特性評価、検出器統合とパッケージング、ビームライン実験を実施。
-   **主な結果**: LaBr3とCeBr3はLYSOよりも約2倍短い減衰時間（約40 ns）を持ち、同等の光収率を維持。これらの材料の実装には、吸湿性、光学的結合、高量子効率などの課題がある。ペロブスカイトや高エントロピー材料などの新興シンチレータクラスも有望な代替案として議論。
-   **意義**: APS-Uのような第4世代シンクロトロンで要求される超高速X線イメージングの性能向上に不可欠なシンチレータ技術の進歩に貢献し、時間分解能の高い材料科学研究を可能にする。
